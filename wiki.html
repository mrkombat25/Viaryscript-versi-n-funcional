🧠 ViaryScript Wiki

Bienvenido a ViaryScript, un lenguaje creativo diseñado para expresar ideas, enseñar programación y construir experiencias interactivas desde cualquier dispositivo.

🚀 ¿Qué es ViaryScript?

ViaryScript es un lenguaje simple y visual que permite definir variables, imprimir mensajes y ejecutar funciones personalizadas. Fue creado por Vicente Dev, fundador de Viary Studios, como una herramienta accesible para aprender, crear y compartir.

📰 Noticias — Agosto 2025

  • ✅ Se lanza la versión beta del motor con soporte para variables personalizadas.
  • 🔒 Nueva regla: el código debe comenzar con Runcode para ejecutarse.
  • 🎬 Se está desarrollando la función Reprudetion.video.archivo.tiempo para reproducir videos desde ViaryScript.
  • 🧪 Playground disponible para probar scripts directamente desde el navegador.
  • 💻 termino de desarrollo de # para agregar un comentario que no se va a ejecutar.

📜 Sintaxis básica

Definir una variable

Hola mundo.jola

Imprimir una variable

Print("jola")

Ejecutar el código

Runcode

🧪 Ejemplo funcional

Runcode
Bienvenidos.mod
Print("mod")

Resultado: Bienvenidos
